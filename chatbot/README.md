# 🧠 Mental Wellness Support Chatbot

An AI-powered mental health support chatbot that provides emotional assistance, tracks wellness metrics, and visualizes trends.

## 🚀 Features
- Dynamic AI responses based on user input
- Mood, stress, anxiety, and sleep tracking
- Sentiment analysis using NLP
- Crisis keyword detection
- Interactive dashboard with charts
- Clean and responsive UI

## 🛠 Tech Stack
- Python (Flask)
- SQLite (Database)
- JavaScript (Frontend)
- Chart.js (Visualization)
- TextBlob (Sentiment Analysis)
- OpenAI API (AI responses)

## 📊 Dashboard
- Stress Trend
- Sleep Trend
- Mood Trend
- Weekly summary insights

## ⚠️ Disclaimer
This chatbot is for emotional wellness support only and not a substitute for professional medical advice.
